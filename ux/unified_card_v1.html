1. SectionLayout 骨架

┌─────────────────────────────────────┐ │ sticky_head (每页重注) │ ← phase_banner + task_list + activity_summary ├─────────────────────────────────────┤ │ status (仅首页) │ ← progress_bar / criteria_panel ├─────────────────────────────────────┤ │ body (主内容、跨页分页) │ ← text / reasoning / plan / 当前 active tool ├─────────────────────────────────────┤ │ appendix (仅末页) │ ← tool_history └─────────────────────────────────────┘
核心契约：
  • sticky_head ≤ 25 节点，超出按 activity_summary → task_list 折叠 fallback。
  • 语义切卡：task 完成 / phase 跳变 / round 跳变 / cycle 跳变时主动 dispatch card_split。
  • tool panel 只展开最新 active，其余折叠（含其他 active）。
