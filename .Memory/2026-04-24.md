### 技术要点
- `collapsible_panel` 是飞书卡片 Schema 2.0 元素，header 支持 markdown，body 支持 elements 列表
- PATCH API 对 collapsible_panel 的支持可能有限，通过 `_collapsible_patch_failed` flag 实现自动回退
- 续接卡片通过 Reply API 在同一话题创建，原地修改 StreamingCard 对象（message_id 指向新卡片）
- `_TOOL_LINE_RE` 正则匹配 `_format_tool_run_line()` 生成的工具摘要行，用于阶段分组
- 所有功能均有独立开关，`card_collapsible_enabled=False` + `card_continuation_enabled=False` 完全回退到当前行为
