## AC18: behaviors-only payload 测试覆盖

**变更范围**：`tests/test_ws_card_action_handler.py`（+133 行）

**实现**：

2. `test_value_dict_behaviors_only_dict_form` — 验证 dict（Mapping）形式的 behavior，无 legacy `action.value` 时正确解析
3. `test_dedup_fingerprint_behaviors_only` — 验证 `dedup_fingerprint()` 在 behaviors-only 模式下正确工作，相同内容产生相同指纹
4. `test_ws_client_process_card_action_behaviors_only` — 验证 `_extract_behavior_value` 被 ws_client 正确导入和使用
5. `test_extract_behavior_value_no_value_attribute` — 验证 `_extract_behavior_value` 处理既没有 value 属性也没有 value key 的边缘情况

**验证**：`uv run python -m pytest tests/test_ws_card_action_handler.py -v` → **15 passed**（原有 10 个 + 新增 5 个）
