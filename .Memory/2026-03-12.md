# 2026-03-12 项目记录

## TTADK 工具选择优化

### 问题
1. `official_cli_disabled` 警告在选择工具后显示，但模型列表正常返回，不影响使用
2. 工具选择卡片显示所有工具，但有些在系统上不存在

### 解决方案

#### 工具可用性检查 (`src/ttadk/manager.py`)
- `_check_tool_available()`: 使用 `shutil.which()` 检查工具是否在系统中存在
- `get_tools(filter_available=True)`: 默认过滤不可用的工具
- 过滤后无可用工具时自动回退到完整列表

#### 警告消息优化 (`src/feishu/handlers/system.py`)
- `handle_select_ttadk_tool()`: 只在模型列表为空或有严重警告时发送警告
- `official_cli_disabled` 不再单独发送（不影响使用）

#### 新增测试 (`tests/test_ttadk.py`)
- `test_get_tools_filter_available`: 测试工具过滤功能
- `test_get_tools_fallback_when_no_tools_available`: 测试回退逻辑
