### 任务描述

### 执行内容
1. **新建 `src/ttadk/model_fetcher.py`**：
   - `TTADKModelFetcher` 类：通过 pty 模拟终端交互获取模型列表
   - `fetch_tool_models()`：获取指定工具的模型列表
   - `_parse_model_selection_menu()`：解析模型选择界面的友好名称
   - `_extract_real_model_name()`：从工具界面提取真实模型名称
   - 内置缓存机制，TTL=300秒

2. **更新 `src/ttadk/models.py`**：
   - `TTADKModel` 添加 `friendly_name` 字段

3. **更新 `src/ttadk/manager.py`**：
   - 集成 `TTADKModelFetcher`
   - 添加工具级模型缓存 `_tool_models_cache` / `_cache_time`
   - `get_models()` 支持 `tool_name` 参数
   - 添加 `invalidate_model_cache()` 方法
   - `_load_models_from_sync()` 接受 `tool_name` 参数
   - 移除未使用的 `_load_models()` 方法

4. **更新 `src/ttadk/__init__.py`**：
   - 导出 `TTADKModelFetcher`

5. **更新 `tests/test_ttadk.py`**：
   - 新增 7 个测试用例：ANSI 清理、菜单解析、模型名称提取、缓存、工具级模型获取

### 技术要点
1. **pty 终端模拟**：使用 `pty.openpty()` 模拟终端交互
2. **ANSI 颜色码处理**：正则表达式清理终端输出
3. **多级缓存**：Manager 层 + Fetcher 层双重缓存
4. **优雅降级**：终端交互失败时回退到默认模型列表
