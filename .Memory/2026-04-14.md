## /help 卡片扁平化重构（全展开 + 手机快捷入口）

### 任务描述
原 `/help` 卡片使用 4 个 tab（编程模式/Deep 任务/项目管理/更多…）切换显示，每次只能看到一个分类。用户要求：一次展开所有命令；如果能提供手机端方便的交互更好。

### 执行内容
重写 `src/card/builders/system.py::SystemBuilder._build_help_card_cached`：

1. **移除 tab 切换** — `category` 参数保留（向后兼容 `handle_help_category` 调用方），但不再驱动分段，每次渲染全部内容。
2. **新增快捷入口按钮行**（手机优先点按，不需输入斜杠命令）：`🧠 Deep 任务 / 🎮 TTADK / 🧩 ACP / 📊 状态 / 🔄 切换项目 / ➕ 新建项目`。所有 action 均复用已在 `ws_client.py` 注册的 callback（`enter_deep_prompt`、`show_ttadk_menu`、`show_acp_menu`、`show_status`、`switch_project`、`new_project_prompt`）。
3. **全部分类内联展开**，6 个 section 之间用 `hr` 分隔：编程模式切换 / Deep Engine / Loop Engine / Spec Engine / 项目管理 / TTADK 管理。底部保留使用提示，新增「手机端优先点按快捷入口」引导。
4. **保持 schema 2.0 + `lru_cache`**，无需改 handler/ws_client，category 参数忽略（通过 `del category` 声明）。

### 技术要点
- 手机端交互走 callback button + responsive layout（>2 按钮自动垂直栈，见 `src/card/shared.py::build_responsive_layout`），无需引入 `collapsible_panel`（兼容性风险）。
- 扁平化后 `_build_help_card_cached` 仍是幂等缓存函数（primitives 参数），`category` 任意值命中同一缓存。
