# GhostAP 项目进展记录

## 项目概述
GhostAP 是一个飞书机器人Shell沙箱服务，通过飞书机器人对话来安全执行本地shell命令，并支持 Coco AI 远程开发模式。

### 支持的功能

#### 📟 Shell 模式（默认）
直接发送 shell 命令执行

#### 🤖 Coco 模式
与 Coco AI 进行远程开发对话：
- 说「帮我写代码」或 `/coco` - 进入 Coco 模式
- 说「退出」或 `/end_coco` - 退出 Coco 模式
- `/coco_info` - 查看会话信息
- 进入 Coco 模式后，消息直接转发给 Coco，不经过 ReAct

#### 📁 目录切换
- 说「切换到xxx目录」- 智能切换工作目录
- 支持自然语言描述：「切换到用户目录下的workspace」

#### 🧠 ReAct 智能意图识别
使用 Ollama 大模型进行推理式意图理解：
- Thought（思考）→ Action（行动）→ Observation（观察）→ Reflection（反思）
- 支持复合意图拆解为多步骤任务
- 例如：「切换到项目目录然后帮我写代码」→ 拆解为 2 个步骤执行

#### 😀 表情回复
- 收到消息：OK 表情
- Coco 处理中：Typing 表情
- 完成：Done 表情
- 多任务执行：Rocket 表情

### 技术栈
- Python 3.11+
- lark-oapi (飞书SDK，长连接模式)
- LangChain + langchain-ollama (AI Agent + ReAct 意图识别)
- pydantic-settings (配置管理)
- coco CLI (远程开发)

### 安全机制
1. 正则表达式检测危险命令模式
2. 命令黑名单配置
3. AI辅助安全检查（可选）
4. 命令执行超时控制
5. 输出长度限制
6. 消息过期丢弃（30秒）

### 连接方式
使用飞书SDK的**长连接模式（WebSocket）**：
- ✅ 无需公网IP或域名
- ✅ 无需内网穿透
- ✅ 本地只要能访问公网就能接收消息
- ✅ 自动加密传输
